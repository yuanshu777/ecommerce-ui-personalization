# Portfolio Presentation Guide

## Best Demo Format for Recruiters
1. Keep a concise README with key numbers near the top.
2. Include one static chart screenshot for immediate signal.
3. Add a short notebook order and reproducibility section.
4. If possible, publish a lightweight dashboard or report page with key outputs.

## Suggested Add-ons
- `results/figures/` with 2-3 exported summary visuals.
- A one-page executive summary (`docs/executive_summary.md`).
- Optional Streamlit dashboard for interactive storytelling.

## Interview Talking Points
- How proxy metrics were chosen without direct purchase labels.
- Why treatment/control was defined this way in production logs.
- How you separate statistical significance from causal confidence.
